Do you need to provide data and documentation?

You need to provide the data and complete this form:

  • If you use any data generated over the course of your NM EPSCoR project in a publication, you need to provide the data and complete this form.
  • If you installed instrumentation in the field for ongoing data collection, such as a weather station or well monitoring equipment, you need to provide that data.
  • If you performed model simulations that were or will be used in a publication, you need to provide the model outputs referenced in that publication. If you'd like, you can also provide the model inputs.
  • If you performed a survey, please make sure that you are providing anonymized data for inclusion in the public data portal.

If you have data such as:

  • text files from a logger
  • text files from a piece of lab equipment
  • raster outputs from a model
  • spreadsheets from anonymized survey data
  • proprietary data files from a piece of equipment
  • a database from a geologic survey
This list is not exhaustive. If you are not sure if you need to deliver data to NM EPSCoR, please contact NM EPSCoR or EDAC for more information.
